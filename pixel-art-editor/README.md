# Pixel Art Editor

A simple browser-based pixel art editor. Big pixels, limited palette, satisfying clicks.

## Features

- 16x16, 32x32, or 64x64 canvas
- 16-color palette
- Brush, eraser, fill bucket, color picker
- Undo/redo (50 steps)
- Grid toggle
- Save as PNG (true pixel size)

## Controls

| Key | Action |
|-----|--------|
| Left Click | Paint/Use tool |
| B | Brush |
| E | Eraser |
| F | Fill bucket |
| P | Color picker |
| G | Toggle grid |
| Ctrl+Z | Undo |
| Ctrl+Y | Redo |

## Run

Just open `index.html` in your browser. No server needed.

```bash
open index.html
```
