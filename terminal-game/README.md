# Terminal Snake Game

Classic Snake game running in your terminal using Python curses.

## Run

```bash
python3 snake.py
```

## Controls

| Key | Action |
|-----|--------|
| Arrow keys | Move snake |
| Q | Quit game |

## Features

- Colorful terminal graphics
- Score tracking
- Increasing speed as you grow
- Wall and self-collision detection
- Game over screen with final score

## How to Play

1. Control the snake with arrow keys
2. Eat the red `*` to grow and score points
3. Avoid hitting walls and yourself
4. Game speeds up as your score increases

## Requirements

- Python 3.x
- curses (included with Python on macOS/Linux)

For Windows, install windows-curses:
```bash
pip install windows-curses
```
